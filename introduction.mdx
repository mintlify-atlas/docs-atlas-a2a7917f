## Overview

Asistencias is a web-based attendance management system designed specifically for municipal internship programs. The system provides role-based access control, real-time attendance tracking, and comprehensive reporting capabilities to streamline intern management.

## Who is this for?

Asistencias is designed for:

- **Municipal HR Departments** managing intern programs
- **Internship Coordinators** tracking attendance and performance
- **Supervisors** monitoring intern schedules
- **Interns** checking in and viewing their attendance records

## Technology Stack

Asistencias is built with:

- **Backend**: PHP with MySQLi
- **Database**: MySQL
- **Frontend**: HTML, CSS, Bootstrap 4.5.2
- **Security**: Password hashing, input sanitization, prepared statements
